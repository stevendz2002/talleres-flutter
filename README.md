## Breve descripción del taller
El objetivo de este taller es practicar el uso de `StatefulWidget` y `setState()` en Flutter para actualizar dinámicamente el estado de la interfaz de usuario. Además, permite implementar widgets adicionales, como `GridView` y `ListView` horizontal, para organizar y mostrar información de manera visualmente atractiva y centrada.

## Pasos para ejecutar la app
1. Clonar el repositorio:
   git clone https://github.com/stevendz2002/talleres-flutter.git

## 2. Cambiar a la rama del taller:

git checkout feature/taller1

## 3. Instalar dependencias:

flutter pub get

## 4. Ejecutar la app:

flutter run
